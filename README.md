# B站视频下载器

基于 Python 的 Bilibili 视频下载工具，提供图形化界面，支持未登录状态下的视频与音频下载。

## 环境要求

- Python 3.8+
- `requests`（HTTP 请求）
- `ffmpeg`（音视频合并与格式转换）

```bash
pip install -r requirements.txt
```

## 启动方式

```bash
python3 bilibili_downloader.py
```

## 功能说明

### 基本下载

- 支持 BV 号、AV 号、b23.tv 短链接
- 自动解析多 P 视频，依次下载每一 P
- 默认选取最高画质的 dash 流，自动合并音视频为 mp4

### 音频相关

| 选项 | 行为 |
|---|---|
| 下载完成后提取音频到 music 目录 | 下载完整视频后，通过 ffmpeg 提取音频为 mp3 |
| 仅下载音频到 music 目录 | 跳过视频流，直接下载音频流并转为 mp3，省时省空间 |

### 缓存管理

| 按钮 | 功能 |
|---|---|
| 清除缓存 | 扫描下载目录中的视频文件，弹窗展示并支持批量删除 |
| 清除音频 | 扫描 `music/` 子目录中的音频文件，弹窗展示并支持批量删除 |

清除缓存时可勾选"同时删除 music 目录中的同名音频文件"，实现视频和对应音频的一并清理。

## 目录结构

```
{保存路径}/
├── video.mp4          # 下载的视频
├── video - P2.mp4     # 多 P 视频的第二 P
└── music/
    └── video.mp3      # 提取或下载的音频
```
