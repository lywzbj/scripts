# scripts

Python 脚本合集，包含 B 站视频下载器和音频播放器。

## 环境要求

- Python 3.8+
- `requests`（HTTP 请求）
- `ffmpeg`（音视频合并与格式转换）

```bash
pip install -r requirements.txt
```

---

## B 站视频下载器

`bilibili_downloader.py` — 基于 Python 的 Bilibili 视频下载工具，图形化界面，支持未登录状态下的视频与音频下载。

### 启动

```bash
python3 bilibili_downloader.py
```

### 基本下载

- 支持 BV 号、AV 号、b23.tv 短链接
- 自动解析多 P 视频，依次下载每一 P
- 默认选取最高画质的 dash 流，自动合并音视频为 mp4

### 音频相关

| 选项 | 行为 |
|---|---|
| 下载完成后提取音频到 music 目录 | 下载完整视频后，通过 ffmpeg 提取音频为 mp3 |
| 仅下载音频到 music 目录 | 跳过视频流，直接下载音频流并转为 mp3，省时省空间 |

### 缓存管理

| 按钮 | 功能 |
|---|---|
| 清除缓存 | 扫描下载目录中的视频文件，弹窗展示并支持批量删除 |
| 清除音频 | 扫描 `music/` 子目录中的音频文件，弹窗展示并支持批量删除 |

清除缓存时可勾选"同时删除 music 目录中的同名音频文件"。

### 目录结构

```
{保存路径}/
├── video.mp4
├── video - P2.mp4
└── music/
    └── video.mp3
```

## 音频播放器

`audio_player.py` — 基于 tkinter + afplay 的本地音频播放器，支持播放列表和歌单管理。

### 启动

```bash
python3 audio_player.py
```

### 播放控制

| 按钮 | 功能 |
|---|---|
| 上一首 / 下一首 | 按列表顺序切换曲目 |
| 播放 / 暂停 / 继续 | 控制当前曲目播放状态 |
| 停止 | 停止播放并重置进度 |

### 播放模式

| 模式 | 行为 |
|---|---|
| 列表循环 | 当前曲目播放完毕后自动播放下一首，末首后回到第一首 |
| 单曲循环 | 当前曲目播放完毕后重新播放 |
| 列表随机 | 随机播放列表中未播放过的曲目，全部轮完一轮后重置 |

### 播放列表

- **添加歌曲**：通过文件对话框批量添加音频文件
- **移除所选**：从列表中移除选中的曲目（支持多选）
- **清空列表**：清空整个播放列表
- 双击列表中的曲目可直接播放

### 歌单管理

- **创建歌单**：将当前播放列表保存为命名歌单
- **加载歌单**：选中歌单后加载到当前播放列表
- **删除歌单**：移除不再需要的歌单
- **保存当前**：快速将当前播放列表保存为"当前列表"
- 歌单数据保存在 `playlists.json` 中
